# 📊 HR Workforce Analytics Dashboard

## 📌 Project Overview

This project showcases an end-to-end **HR Workforce Analytics** solution using **Python, SQL, and Power BI**. 
The workflow includes data cleaning and exploratory data analysis in Python, SQL-based business analysis, 
and the development of an interactive Power BI dashboard to monitor employee performance, salaries, work modes, hiring trends, and workforce distribution.

---

## 🎯 Project Objectives

* Clean and preprocess HR employee data using Python.
* Perform business analysis using SQL.
* Build an interactive Power BI dashboard.
* Analyze employee distribution, salaries, performance, and hiring trends.
* Deliver actionable insights to support HR decision-making.

---

## 🛠️ Tools & Technologies

* Python (Pandas, NumPy)
* Matplotlib 
* SQL (MySQL)
* Power BI
* Power Query
* DAX

---

## 📂 Dataset

The dataset includes the following employee attributes:

* Employee_ID
* Full_Name
* Department
* Job_Title
* Hire_Date
* Hire_Year
* Location
* Performance_Rating
* Experience_Years
* Status
* Work_Mode
* Salary_INR

---

## 🧹 Data Cleaning & EDA (Python)

* Removed duplicate records.
* Handled missing values.
* Standardized categorical values.
* Converted data types.
* Formatted date columns.
* Performed Exploratory Data Analysis (EDA).
* Created visualizations using Matplotlib and Seaborn.

---

## 🗄️ SQL Analysis

The following business analyses were performed using SQL:

* Retrieved complete employee records.
* Calculated the total number of employees.
* Analyzed employee distribution by department.
* Examined workforce distribution by work mode.
* Evaluated employee status (Active/Inactive).
* Calculated average salary for each department.
* Identified the top 10 highest-paid employees.
* Analyzed performance rating distribution.
* Measured average experience across departments.
* Tracked employee hiring trends by year.
* Identified top-performing employees (Performance Rating ≥ 4).
* Calculated department-wise active employees.
* Retrieved employees working in remote mode.
* Identified employees with more than 5 years of experience.
